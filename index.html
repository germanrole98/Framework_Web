Firma de abogados “Te apoyamos SAS”,

Análisis del problema

La firma “Te Apoyamos SAS” necesita una aplicación web que le permita:

  1. Mostrar información institucional (historia, servicios, noticias, profesionales, alianzas, contacto).
  2. Recibir y evaluar hojas de vida de profesionales interesados en trabajar con la firma.
  3. Gestionar los procesos legales de los clientes con registro de datos y avances.

La solución debe cumplir con estándares de seguridad, confidencialidad y respaldo de datos debido a la sensibilidad de la información como datos personales y datos de los procesos legales que se llevan a cabo en la firma.

Requerimientos

Funcionales

  • Publicar información de la firma y sus servicios.
  • Incluir un formulario de contacto.
  • Permitir que los profesionales externos registren su hoja de vida.
  • Validar y comunicar si el postulante es aceptado o no.
  • Registrar clientes y procesos legales con sus avances.
  • Acceso diferenciado según el tipo de usuario (administrador, profesional, postulante, público).

No funcionales

  • Seguridad en el manejo de datos.
  • Facilidad de uso.
  • Disponibilidad en internet 24/7.
  • Accesible desde navegador web y móvil.

Actores involucrados

  • Administrador de la firma: gestiona usuarios, información y procesos.
  • Profesionales internos: llevan el registro y actualización de procesos.
  • Postulantes externos: registran su hoja de vida y reciben respuesta.
  • Público en general: consulta información sobre la firma y servicios.

Información a registrar

  • Firma: trayectoria, servicios, noticias, reconocimientos, alianzas, formas de contacto.
  • Profesionales: datos básicos, experiencia, perfil general.
  • Postulantes: datos personales, hoja de vida, estado de evaluación.
  • Clientes: información de contacto.
  • Procesos legales: tipo de proceso, estado y avances.

Frameworks para Páginas Web Responsive

Bootstrap

Uno de los frameworks CSS más populares. Ofrece un sistema de grid responsive y componentes listos para usar como botones, menús, formularios y muchos mas componentes. Ideal para proyectos rápidos.

  • Fácil de aprender.
  • Gran cantidad de documentación y comunidad activa.
  • Compatible con la mayoría de navegadores.

Visita bootstrap.com

Tailwind CSS

Framework CSS basado en clases utilitarias, que permite crear diseños personalizados sin escribir mucho CSS. Ideal para desarrolladores que buscan un nivel de personalización y control mas amplio en el diseño

  • Diseño muy flexible y moderno.
  • Permite optimización en el tamaño final del CSS.

Visita tailwindcss.com

Bulma

Framework CSS ligero, moderno y 100% responsive basado en Flexbox. Ideal para web limpias y minimalistas.

  • Sintaxis simple.
  • No requiere JavaScript para sus componentes básicos.

Visita bulma.io

Materialize

Framework basado en Material Design de Google. Ideal para proyectos que buscan seguir las guías de diseño de Google.

  • Interfaz moderna con animaciones.
  • Compatible con dispositivos móviles.

Visita materializecss.com

Astro

Framework para construir sitios estáticos y rápidos con enfoque en optimización y diseño responsivo. Ideal para blogs, portafolios y landing pages.

  • Arquitectura ligera con poco JavaScript.
  • Integración con otros frameworks como React y Vue.

Visita astro.build
